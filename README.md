<h1>Unix Sockets</h1>

<p>Unix socket server with multiple clients</p>

<h1>Getting Started</h1>

install:
<p>composer require maxfolder/unix_socket</p>

<p>set path to you socket file in socket.ini </p>

example: /var/www/server.socket
